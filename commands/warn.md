# Detailed info about /warn command

## /warn add

### Arguments
1. User (Required) - Tag user you want to take action against.
2. WarnID (Required) - Specify a warn ID (of user) to remove.

### Permissions required
1. MODERATE_MEMBERS


## /warn remove

### Arguments
1. User (Required) - Tag user you want to take action against.
2. Reason (Not required) - Specify a reason of your action.

### Permissions required
1. MODERATE_MEMBERS
